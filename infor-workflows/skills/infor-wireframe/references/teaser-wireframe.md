# Teaser Wireframe

Slide-by-slide wireframe for an INFOR Financial Group teaser. Use together with
[`SKILL.md`](../SKILL.md). All copy is drafted via
[`infor-deck-writing`](../../infor-deck-writing/SKILL.md) (see "Teaser — Cover line",
"Teaser — The Opportunity (page 1)", and "Teaser — Page 2" recipes). All layout, branding, fonts,
colors, and template scaffolding are governed by
[`brand-guidelines-infor`](../../brand-guidelines-infor/SKILL.md).

---

## Header / Defaults

| Item | Default |
|------|---------|
| Page count | **2 pages** (standard) — every example teaser in the INFOR corpus is exactly 2 pages |
| Orientation | 10" × 7.50" widescreen 4:3, same as the rest of the INFOR corpus |
| Anonymisation | Use a **Project codename** ("Project Daytona", "Project Lightning", "Project Meridian", "Project Thunderbird") and **a defined short form** for the Company. Real legal names appear only when the Company has authorised an un-anonymised teaser (e.g., Faba Canada) |
| Header stamp | A short calendar stamp in the top-right of every slide — `Winter 2026`, `Fall 2024`. Not a year on its own |
| Confidentiality stamp | `Strictly Private & Confidential` (top right) on every slide |
| Disclaimer | The standard INFOR teaser disclaimer goes at the **bottom of page 2** (verbatim — see `infor-deck-writing` Section 4). It is **not** a separate slide on a teaser |
| Contact block | Inline on page 2 in a 4-row Key Contacts table — not a separate "Contact" slide |

The teaser is the only INFOR deliverable type that does **not** use the standard
[Cover → Executive Summary → Content → Disclaimer → Contact](../../brand-guidelines-infor/SKILL.md)
scaffolding. Page 1 carries the engagement / company description (no separate cover); page 2
carries the financials, contacts, and the disclaimer (no separate contact / disclaimer page).
This is a deliberate exception to `brand-guidelines-infor` Section 11.6, observed across every
teaser in the corpus.

## Per-Slide Notes

### Page 1 — The Opportunity

- **Positioning headline (sub-title under the project codename)** — a 6–14 word noun phrase
  describing the asset (`Leading B2B SaaS Digital Wellbeing Platform`, `Specialty Automotive
  Services Franchisor`, `AI-Driven Legal Platform with Rare Caselaw Backfile Upside`). Drafted via
  the `infor-deck-writing` recipe "Teaser — Cover line".
- **Engagement line** — opens with the defined Company term, names INFOR as exclusive financial
  advisor, names the Transaction type (sale / majority divestiture / minority capital raise of
  C$XXMM / refinancing / strategic review). Verbatim skeletons live in `infor-deck-writing`
  "Teaser — The Opportunity (page 1)".
- **Key Figures tiles** — 5–6 tiles is the corpus norm. Each tile is a large bold figure
  (`~C$46MM`, `+13MM`, `~90%`, `8.0%`, `60,000+`) with a 2–3-word caption below. Use the figure
  to carry the unit (`~`, `C$`, `+`, `%`).
- **Investment Highlights** — 4–6 items is standard (5 is the median across the corpus). Each
  item: bold positioning headline + 1–3 supporting sub-bullets. Use the positioning-headline
  pattern menu in `infor-deck-writing`.
- **Founder / management call-out** — Situational. Include when the founder is a meaningful part
  of the story (Faba Canada). Otherwise omit and use the freed space for a sixth investment
  highlight or a larger KPI tile band.
- **Anonymised vs. un-anonymised** — Situational. If the Company is anonymised, the cover line is
  the Project codename + positioning headline. If un-anonymised (less common — e.g., Faba Canada),
  the company name and small logo appear on the page.

### Page 2 — Financial Summary, Features, Contacts, Disclaimer

- **Chart placement** — Always include 3–4 financial / operational chart tiles. Use placeholder
  rectangles in the wireframe; the analyst pastes Excel-generated charts in at construction time
  (see `brand-guidelines-infor` §11.5 on placeholder rectangles).
- **Platform Core Features / Service Highlights** — Situational on the exact label. Use
  `Platform Core Features` for SaaS / tech companies, `Service Highlights` or `Capabilities` for
  services / industrial. 4–6 blocks is standard.
- **Key Contacts table** — 4 INFOR bankers is the corpus norm. Include name, title, phone, and
  email. Format per `brand-guidelines-infor` §7 (tables).
- **Disclaimer block** — Verbatim from `infor-deck-writing` Section 4. Do not paraphrase. Set in
  7 pt Palatino Linotype across the full width of the slide at the bottom, with a thin separator
  line above it.

## Variants

### Slim teaser (1 page)

Rare in the INFOR corpus. Collapses page 2 onto page 1 by dropping the chart tiles and the
features grid, retaining only the Key Contacts row + disclaimer footer. Use only when the user
explicitly asks for a 1-pager and the data story doesn't need charts (e.g., a very early-stage
asset where financials would be misleading).

### Expanded teaser (3–4 pages)

Also rare. Adds a third page for **either** (a) Market / Industry overview (size, growth, drivers)
or (b) Customer-and-product detail. The disclaimer always stays on the **final** page. Use when
the buyer universe is non-obvious or the market thesis is the main selling point.

## Confidence Notes

- **Page count of exactly 2** — Strongly supported (5/5 example decks).
- **Header stamp + confidentiality stamp pattern** — Strongly supported (5/5).
- **Page 1 = engagement + company overview + KPI tiles + investment highlights** — Strongly
  supported (5/5, with minor variation in tile / highlight counts).
- **Page 2 = charts + features + contacts + disclaimer** — Strongly supported (5/5, with the
  exact mix of charts vs. features varying by sector).
- **No separate Cover / Disclaimer / Contact slides** — Strongly supported (5/5). This is the
  one INFOR deliverable that breaks the standard 5-slide scaffolding.
- **Founder call-out on page 1** — Situational; observed in some but not all teasers.

## Handoff

Hand off to [`infor-deck-writing`](../../infor-deck-writing/SKILL.md) for every text element on
the teaser — the cover line, engagement line, company overview paragraph, KPI tile captions,
investment-highlight headlines + sub-bullets, feature-block headlines + descriptions, and the
verbatim disclaimer. Then hand off to
[`brand-guidelines-infor`](../../brand-guidelines-infor/SKILL.md) to construct the 2-page deck
from the INFOR Deck Template, applying brand colors, fonts, and layout.
